Sources, preprocessing pipeline, and the unified dataset

Data Sources

We integrate four official Turkish data sources:

# Source Series Coverage
1 TCMB Konut Fiyat Endeksi Nominal and real KFE, NUTS-2 regional KFE Monthly, 2010-2024
2 TCMB EVDS – Consumer Prices CPI index for deflation Monthly, 2010-2024
3 TCMB EVDS – Exchange Rates USD/TRY monthly average Monthly, 2010-2024
4 TUIK Insaat ve Konut Istatistikleri Housing sales, building permits Annual, 2010-2024

Direct links:

  • TCMB KFE: https://www.tcmb.gov.tr/wps/wcm/connect/tr/tcmb+tr/main+menu/istatistikler/reel+sektor+istatistikleri/konut+fiyat+endeksi/
  • TCMB EVDS (CPI, FX): https://evds2.tcmb.gov.tr
  • TUIK Housing Statistics: https://data.tuik.gov.tr/Kategori/GetKategori?p=insaat-ve-konut-116
Note

Why this dataset? Turkey’s housing market has undergone one of the world’s most extreme price cycles in 2020-2024, driven by a unique combination of currency collapse, triple-digit inflation, post-earthquake displacement, and supply-side contraction. This makes it an exceptionally rich case for understanding how macroeconomic shocks transmit into asset markets.

Preprocessing Pipeline

Raw TCMB and TUIK data require aggregation and cleaning. The full pipeline is in R/01_preprocess.R. Key steps:

Step 1 – Manual extraction. Series were extracted from TCMB publications and TUIK statistical tables into data/raw_*.csv files, preserving original source values.

Step 2 – Tidying. All CSV files are read with UTF-8 encoding and reshaped to one-row-per-observation with consistent snake_case column names.

Step 3 – Derived variables.

Variable Object Formula
kfe_nominal_yoy annual_macro (kfe_nominal / lag - 1) x 100
kfe_real_yoy annual_macro (kfe_real / lag - 1) x 100
cum_2020_2024_pct regional_nuts2 Compounded 5-year nominal change
months_100m2 affordability already in source data
Show code 
# R/01_preprocess.R -- run once before rendering
library(readr); library(dplyr)

rd <- function(f) read_csv(file.path("data", f),
                            show_col_types = FALSE,
                            locale = locale(encoding = "UTF-8"))

monthly_data <- rd("raw_monthly_kfe_cpi_usdtry.csv")

annual_macro <- rd("raw_annual_macro.csv") |>
  mutate(
    kfe_nominal_yoy = (kfe_nominal_2010base / lag(kfe_nominal_2010base) - 1) * 100,
    kfe_real_yoy    = (kfe_real_2010base    / lag(kfe_real_2010base)    - 1) * 100
  )

regional_nuts2 <- rd("raw_regional_nuts2.csv") |>
  mutate(
    cum_2020_2024_pct =
      ((1 + kfe_2020_yoy_pct/100) * (1 + kfe_2021_yoy_pct/100) *
       (1 + kfe_2022_yoy_pct/100) * (1 + kfe_2023_yoy_pct/100) *
       (1 + kfe_2024_yoy_pct/100) - 1) * 100
  )

city_monthly   <- rd("raw_city_kfe_monthly.csv")
affordability  <- rd("raw_affordability_annual.csv")
quarterly_fx   <- rd("raw_quarterly_kfe_fx.csv")
housing_sales  <- rd("raw_housing_sales_annual.csv")
permits        <- rd("raw_building_permits_annual.csv")
minwage        <- rd("raw_minimum_wage_history.csv")
kfe_new_exist  <- rd("raw_kfe_new_vs_existing.csv")

save(monthly_data, annual_macro, regional_nuts2, city_monthly,
     affordability, quarterly_fx, housing_sales, permits,
     minwage, kfe_new_exist,
     file = "data/housing_market_tr.RData")

Data Quality Notes

  • KFE methodology revision (July 2024). TCMB revised the KFE from a quarterly to a monthly hedonic regression model, rebasing to 2023=100. The monthly series here preserves both bases (2023=100 and 2010=100) for comparability.

  • Regional data. NUTS-2 regional breakdowns (26 regions) are published as annual % change figures. Median unit prices are TCMB quarterly figures averaged to annual.

  • Sales data. First-hand / second-hand split is available from 2015. 2024 total reflects TUIK revised figure of 1,319,539.

  • CPI deflation. Real KFE uses TURKSTAT official CPI (2003=100 base, consistent series converted to 2010=100 for this analysis).

  • Missing values. Some regional 2024 figures are NA where TCMB had not published quarterly data by the time of data collection. These are coded as NA, not imputed.
